Proyecto

Este proyecto es una aplicación web de búsqueda y gestión de usuarios, desarrollada como trabajo final de la materia Big Data. Integra MongoDB y Elasticsearch para almacenar, indexar y consultar grandes volúmenes de documentos, además de administrar usuarios y sus permisos dentro de la plataforma. Permite cargar datos desde archivos ZIP o mediante web scraping y explorarlos a través de un buscador especializado. Su objetivo es demostrar, de forma práctica, cómo las tecnologías de Big Data pueden apoyar procesos reales de consulta y toma de decisiones.
